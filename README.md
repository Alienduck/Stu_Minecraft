# Stu_Minecraft
Minecraft in Rust ???

# How to Play ?

If you want to play you can download the release or clone/fork this repo and do this command:
```powershell
cargo run --release
```
*warn: the compilation can take a long (like ~5min) but this is for the first compilation, then it only take a few seconds*

Then the game should appear, if you have any problem head to [issues](https://github.com/Alienduck/Stu_Minecraft/issues)
